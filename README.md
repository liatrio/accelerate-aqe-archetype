# Archetype

## Installation

```
mvn install
```


```
mvn archetype:generate -DgroupId=com.liatrio.accelerate -DartifactId=accelerate-aqe-archetype -DarchetypeArtifactId=maven-archetype-archetype

mvn archetype:generate                                  \
  -DarchetypeGroupId=com.liatrio.accelerate                \
  -DarchetypeArtifactId=accelerate-aqe-archetype          \
  -DarchetypeVersion=1.0-SNAPSHOT                \
  -DgroupId=<my.groupid>                                \
  -DartifactId=<my-artifactId>

```
